## Image Guidelines

### Recommended Specifications
- **Hero Images**: 1920x1080px or larger, high quality
- **Service Images**: 800x600px minimum
- **Portfolio Thumbnails**: 600x400px (3:2 aspect ratio)
- **Project Details**: 1200x800px minimum

### Format Recommendations
- **Photos**: JPG (85-90% quality)
- **Graphics/UI**: PNG (for transparency) or WebP
- **Optimize all images** for web (use tools like TinyPNG, Squoosh, or ImageOptim)

### Color Treatment
- Most images should work well with dark backgrounds
- Consider adding subtle blue glow overlays for key images
- Maintain contrast for readability

## Placeholder Images

## To Add Images

1. Place images in this directory (`assets/images/`)
2. Use the exact filenames referenced in the HTML files
3. Optimize images before adding to reduce load times
4. Consider using WebP format with JPG/PNG fallbacks for best performance
